# 现有页面与 Axure 原型对应清单

原型地址：<https://www.axureshop.com/ys/1496238>

说明：

- “独立页面”表示当前路由与原型页面结构基本一致。
- “同页弹窗”表示业务已实现，但新增、编辑或详情仍在列表页弹窗中，与原型的独立页面不同。
- “合并标签页”表示原型拆成多个页面，当前项目为了快速完成将其合并为一个带标签页的页面。
- 本清单推荐从上到下逐页检查；发现样式差异时记录“路由 + 截图 + 期望效果”。

## 弹窗改独立页面的建议顺序

优先转换业务最强、答辩最容易被检查的页面：

1. 销售单新增、销售单详情。
2. 采购单新增、采购单详情。
3. 销售退货和采购退货的新增、详情。
4. 库存盘点、调拨、报损、借入借出的新增和详情。
5. 收款单、付款单的新增和详情。
6. 商品、客户、供应商的新增和详情。
7. 员工、角色的新增页面。

配置类资料（分类、单位、标签）字段少，保留弹窗不会明显影响业务演示，可最后处理。
